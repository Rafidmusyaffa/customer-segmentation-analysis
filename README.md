# Customer Segmentation Analysis

## Project Overview
This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis to identify different types of customers and their purchasing behavior.

Customer groups were discovered using K-Means clustering after feature scaling.

## Objectives
- Understand customer purchasing behavior
- Identify high-value customers
- Segment customers for targeted marketing strategies

## Methodology

1. Data Cleaning
2. Feature Engineering (RFM Metrics)
3. Feature Scaling
4. Customer Segmentation using K-Means Clustering
5. Visualization and Business Insights

## Key Insights

- VIP customers contribute the majority of revenue.
- Loyal customers purchase frequently but spend slightly less per transaction.
- At-risk customers show declining purchasing activity.
- Low-value customers contribute minimal revenue.

## Business Recommendations

VIP Customers:
Implement loyalty programs and exclusive rewards.

Loyal Customers:
Upsell premium products and cross-sell complementary items.

At-Risk Customers:
Use targeted promotions to re-engage them.

Low Value Customers:
Automate marketing campaigns to increase engagement.
